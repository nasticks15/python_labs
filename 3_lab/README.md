# Звіт до роботи
## Тема: Вступ у тестування
### Мета роботи: Ознайомитись із механізмом внутрішніх перевірок у Python за допомогою assert, навчитися використовувати їх для валідації даних та контролю правильності виконання програмних блоків, підготуватися до юніт-тестування.


---
### Виконання роботи
1. Використання assert для внутрішніх перевірок
2. Валідація відповіді від LLM
3. Санітизація введених даних
4. Контроль змінюваних об’єктів
5. Строга типізація у функціях
6. Порівняння assert та if
7. Використання власних помилок
